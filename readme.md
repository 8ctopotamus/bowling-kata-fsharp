# F# Bowling Kata

**[Bowling Kata](https://codingdojo.org/kata/Bowling/)** in `F#`.

Run the game: 

`dotnet run`

Run in "interactive" mode (human player): 

`dotnet run interactive`

## Dependencies
* Dotnet 6
